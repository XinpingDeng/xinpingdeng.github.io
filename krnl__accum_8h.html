A overloaded template kernel to.

  1. accumulate components of each complex number
  2. accumulate powr of 2 of components of each complex number

The accumulation can happens with history.

The accumulation happens inside individual polarisation of each antenna. The input of this kernel is the output of a reverse FFT, which generat fine sampled time series with fine channels. The output of this kernel will be used to calculate scaling number so that we can scale reverse FFT output from 16-bits or 32-bits float to 2, (no 4 for now) 8 or 16-bits integers. The scaling happens within individual polarisation of each antenna as well.

Currently it works with a single coarse channel, multiple antennas and 2 polarisations, we may need to extend it to support multiple coarse channels in future.
